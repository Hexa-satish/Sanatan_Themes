# Sanatan Keyboard Themes

Public theme assets for Sanatan Keyboard.

## Structure

```
themes/
  theme_01/
    album.png          # grid thumbnail (ThemeFragment)
    preview.png        # apply-screen preview (ThemeApplyActivity)
    keyboard_image.png # keyboard background
    colorcode.png      # text color hint
    mdpi|hdpi|xhdpi|xxhdpi/
      key_presed.9.png
      key_unpresed.9.png
      popup_bg.9.png
  theme_02/ ... theme_15/
themes.json
```

Push to GitHub as public repo. Update `base_url` in `themes.json` if username differs.
